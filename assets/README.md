# ASSETS — INTIS Web

Coloque en esta carpeta los siguientes archivos antes de publicar:

## Logo
- `logo.png`        — Logo principal (fondo transparente o el logo provisto)
                      Recomendado: mínimo 300x300px, PNG con transparencia

## SEO / Compartir en redes
- `og-image.png`    — Imagen para Open Graph (compartir en redes sociales)
                      Recomendado: 1200x630px
                      Puede ser una imagen del logo sobre fondo azul marino corporativo

## Favicon (opcional, para mayor compatibilidad)
- `favicon.ico`     — Favicon clásico 32x32px
- `favicon-192.png` — Para Android/PWA 192x192px
- `favicon-512.png` — Para PWA 512x512px

---
Recuerde actualizar las rutas en index.html, politica-privacidad.html
y terminos-condiciones.html si cambia los nombres de archivo.
